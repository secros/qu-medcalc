MELD 3.0

Model for End-Stage Liver Disease — updated 2021 formula incorporating sex and albumin.

Formula — Kim et al., Gastroenterology 2021
MELD 3.0 = 1.33×(Female) + 4.56×ln(Bilirubin) + 0.82×(137−Na) − 0.24×(137−Na)×ln(Bilirubin) + 9.09×ln(INR) + 11.14×ln(Creatinine) + 1.85×(3.5−Albumin) − 1.83×(3.5−Albumin)×ln(Creatinine) + 6

Score range 6–40. Creatinine capped at 3.0, min 1.0 mg/dL. Albumin constrained 1.5–3.5 g/dL. Na constrained 125–137 mEq/L.

Child-Pugh Score

Estimates cirrhosis severity and 1-year/2-year survival. Based on 5 clinical parameters.

Scoring
Sum of 5 parameters, each scored 1–3. Total 5–15.
Class A (5–6): ~100%/85% 1yr/2yr survival  |  Class B (7–9): ~80%/60%  |  Class C (10–15): ~45%/35%

VOCAL-Penn

Cirrhosis surgical risk score estimating 30-day post-operative mortality. Mahmud et al., Gastroenterology 2021.

Formula — Mahmud et al., Gastroenterology 2021
Linear predictor = −5.111 + 0.848×ln(Bilirubin) + 0.949×ln(INR) + 0.398×ln(Creatinine) + 0.033×Age + 0.527×ASA class + 0.576×Emergency + 0.396×Bleeding − 0.419×Albumin

Mortality % = 100 × eLP / (1 + eLP)
